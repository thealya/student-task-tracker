# Student Task Tracker

## Project Description
Student Task Tracker is a simple web application that helps students manage their tasks and assignments.

## Features
- Add new tasks
- Update task status
- Delete tasks
- Set due dates
- Set task priority
- Store data using Supabase

## Technologies Used
- Vite
- JavaScript
- Supabase
- HTML
- CSS

## How to Run the Application

1. Install dependencies:
   npm install

2. Start the application:
   npm run dev

3. Open the following URL in your browser:
   http://localhost:5173
